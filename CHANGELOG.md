- LinkedIn Easy Apply automation
- AI-powered question answering with RAG

- Support for external platforms (Greenhouse, Ashby)

### Features
- Automated job application process
- Intelligent form filling
- Job fit evaluation
- Resume customization per job posting
- Multi-platform support
- RAG-powered context optimization
